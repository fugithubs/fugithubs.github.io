标题

在Markdown中，可以使用 # 符号表示标题，一级标题使用一个 # 符号，二级标题使用两个 # 符号，依次类推，最多支持六级标题

效果如下：

我是一级标题

我是二级标题

我是三级标题

我是四级标题

我是五级标题
我是六级标题

文本

格式化效果：
加粗：使用 **将文本包围，例如 **加粗**
斜体：使用*将文本包围，例如*斜体*
黑体加斜体：使用***将文本包围，例如***黑体加斜体***
删除线：使用~~将文本包围，例如~~删除线~~
代码：使用 `` 将文本包围，例如 代码

无序列表

使用- 或 * 符号表示无序列表，语法：-（或者*）+空格+列表内容

效果：

  • 无序列表1
  • 无序列表3
  • 无序列表2
  • 星星无序1
  • 星星无序2
  • 星星无序3

有序列表

使用数字表示有序列表，语法：数字+.+空格+列表内容

效果：

  1. 有序1
  2. 有序2
  3. 有序3
